# 🚀 LifeOS – Smart Daily Productivity Web Application

## 🧩 Overview

LifeOS is a modern, full-stack web application designed to help individuals stay organized, focused, and productive throughout their day. It integrates essential productivity tools such as task management, focus tracking, habit monitoring, and meal suggestions into a single, intuitive platform.

This project is built as a **responsive, cloud-enabled web application**, allowing users to securely access their data from any device.

---

## 🎯 Objective

The goal of LifeOS is to provide a user-friendly system that helps users:

- Plan and prioritize daily tasks  
- Maintain focus using time management techniques  
- Track habits and build consistency  
- Receive quick suggestions for daily decisions  
- Stay motivated and structured throughout the day  

---

## ⚙️ Features

### 🔐 User Authentication
- Secure login and signup
- Session management
- Personalized user data

### 📋 Daily Task Planner
- Add, edit, delete tasks
- Mark tasks as completed
- Track progress in real-time
- Priority-based organization

### ⏱️ Focus Timer (Pomodoro)
- Start, pause, reset timer
- Improves concentration and productivity

### 🍽️ Meal Suggestion Tool
- Random meal recommendations
- Categories: Healthy, Quick, Budget-friendly

### 🔁 Habit Tracker
- Track daily habits
- Completion indicators
- Streak tracking

### 😊 Mood & Motivation
- Daily mood logging
- Motivational quotes

### 📊 Dashboard
- Centralized overview of tasks, habits, and progress
- Clean card-based UI
